# ChatGPT突然发不出去消息了？按此教程完美解决

# 现象原因

ChatGpt消息突然发不出去了，历史对话消息是可以看到的，但是新的问题消息发不出去...

如果你的网络没问题，那么猜测可能是下面的原因

应该都是点击登录后顶部弹窗邀请[加入多语言 alapha 测试]了，并且语言选择了中文，抓包看到ab.chatgpt.com接口有问题，控制台报错，拿不到 account信息。

# 解决方案

临时解决方案：浏览器打开无痕隐私模式打开ChatGpt

永久解决方案：清理掉ChatGpt本地缓存，等于还原到最新，清空 https://chat.openai.com/的浏览器本地所有缓存数据。

流程:ChatGPT对话页面-鼠标右键-检查(打开 devtools面板)-应用(顶部)-左侧应用下的存储-勾选「包括第三方cookies」-点击「清除网站数据」
